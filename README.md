# Меню‑Мастер 3000 — Telegram бот для планирования домашнего питания

## Возможности
- Составляет меню (день/неделя/месяц)
- Подбирает рецепты по продуктам
- Формирует списки покупок
- Учитывает диеты, аллергии, бюджет
- Помнит предпочтения пользователей (SQLite)
- Работает через вебхук (идеально для Railway)
- Общается с лёгким юмором
